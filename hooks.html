WordPress Actions aur Filters

1. Hooks Kya Hote Hain?

WordPress mein hooks ek aisi technique hai jisme aap WordPress ke core ya plugin/theme ke beech kisi specific point par apni functions chala sakte ho.

Hooks do tareeqay ke hote hain:

  • Actions
  • Filters

2. Actions Kya Hain?

  • Actions wo hooks hain jahan aap WordPress ko kehte ho:
    ‚ÄúJab ye kaam ho raha ho, toh mera function bhi chalao‚Äù
  • Aap add_action() se apna function attach karte ho, aur do_action() se wo action fire hoti hai.
  • Actions koi value return nahi karte, balki extra kaam perform karte hain ‚Äî jaise email bhejna, log likhna, ya koi additional HTML print karna.

Example:

// Function attach karna (add_action)
add_action( 'after_post_created', 'send_post_email' );

function send_post_email() {
    // Email bhejne ka code yahan hoga
    echo "Email sent!";
}

// Action fire karna (do_action)
do_action( 'after_post_created' );

Key Points for Actions:

  • add_action(): Function ko hook ke sath attach karta hai.
  • do_action(): Hook ko fire karta hai aur sab attached functions chalata hai.
  • Actions me original kaam override nahi hota, bas extra kaam joda jata hai.

3. Filters Kya Hain?

  • Filters ka kaam hota hai kisi data/value ko modify karna ya override karna.
  • Aap add_filter() se function attach karte ho, aur apply_filters() se filter ko apply karte ho.
  • Filters value ko le kar, usko modify kar ke return karte hain.

Example:

// Function attach karna (add_filter)
add_filter( 'product_price', function( $price ) {
    return $price - 10; // 10 rupees discount
});

// Filter apply karna (apply_filters)
$price = apply_filters( 'product_price', 100 );
echo $price; // Output: 90

Key Points for Filters:

  • Filters value ko override kar sakte hain.
  • Filters me hamesha value return karni hoti hai.
  • Filters ko use karke aap WordPress ke default behavior ko customize kar sakte hain.

5. Real World Scenario

Maan lo Archive page pe aapka data grid mein show ho raha hai.

  • Aap do_action( 'my_custom_archive_grid', $args ) likhte ho.
  • Phir add_action( 'my_custom_archive_grid', 'render_archive_grid', 10, 1 ) se us action ke sath apna function attach karte ho, jo data query karta hai aur grid ka markup banata hai.
  • Jab AJAX request aati hai with filters (like category), toh aap do_action() ke zariye wohi rendering logic dubara chala ke filtered HTML wapas bhejte ho.

6. Filters ke zariye Markup ko Customize Karna

Aap markup ko direct add_action() mein generate kar sakte ho, magar agar aap chahte ho ke user us markup ko customize kar sake, toh aap apply_filters() ka use karte ho.

function render_archive_grid( $args ) {
    $markup = '<div class="grid-item">Product Title</div>';
    $markup = apply_filters( 'custom_grid_item_markup', $markup, $args );
    echo $markup;
}
add_action( 'my_custom_archive_grid', 'render_archive_grid', 10, 1 );

User apni marzi se markup change kar sakta hai:

add_filter( 'custom_grid_item_markup', function( $markup, $args ) {
    return '<div class="custom-grid-item">Apni Custom Title</div>';
}, 10, 2 );

7. Summary

  • Action: Extra kaam karwana, original kaam ko override nahi karta.
  • Filter: Value modify ya override karta hai.
  • add_action() aur add_filter() se aap apne functions ko hooks ke sath attach karte hain.
  • do_action() aur apply_filters() se aap hooks ko trigger karte hain.
